# Proyecto: Análisis de Ventas E-commerce (SQL)

## Descripción
Este proyecto consiste en el diseño y análisis de una base de datos relacional para una tienda online. El objetivo es transformar datos transaccionales brutos en información estratégica para la toma de deciones utilizando **MySQL**

---

## Métricas Clave y Business Insights
A través de los scripts de este repositorio, se extraen los siguientes indicadores de negocio:

**Customer Lifetime Value (LTV)**: Identificación de los clientes con mayor gasto acumulado para estrategias de marketing. 

**Product Performance**: Análisis de los siguientes productos más vendidos para optimizar la gestión de inventario. 

**Tendencia de Facturacion**: Seguimiento de ingresos mensuales para detectar estacionalidad y picos de venta. 

---

## Contenido del Proyecto
**Sales_analysis.sql**: Script para la creación de la base de datos, arquitectura de tablas y carga de datos de ejemplo. 

**queries_analysis.sql**: Consultas avanzadas para el análisis de los KPIs mencionados anteriormente. 

**README.md**: Documentación completa del proyecto. 

---

## ``Cómo Ejecutar``
1. **Preparación**: Asegúrate descargar los archivos y tener Instalado MySQL. 
2. **Configuración**: Ejecutar el script en MySQL
3. **Análisis**:Ejecutar consultas desde queries_examples.sql para análisis.

---

## Consultas de análisis incluidas
  -  Ventas totales por cliente.
  - Productos más vendidos. 
  -  Ingreso por mes.
